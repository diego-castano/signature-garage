# Cambios realizados - 5 de Marzo 2026

## Velocidad del sitio

- La pagina ahora carga en 1.6 segundos (antes tardaba 20.6 segundos)
- Todas las imagenes se convirtieron a formato WebP, que pesa mucho menos
- Las imagenes se cargan solo cuando el usuario baja en la pagina (lazy loading)
- 6 archivos de estilos ahora cargan sin bloquear la pagina
- Se activo cache del navegador: las visitas repetidas cargan mucho mas rapido
- Se activo compresion Gzip para que los archivos viajen mas livianos
- Se eliminaron scripts de emojis innecesarios (~45KB menos)

## Imagenes optimizadas

- Se procesaron las 2,003 imagenes de la biblioteca de medios
- Se optimizaron 34 imagenes del tema (de 9.2 MB a 1.4 MB en WebP)
- Cada imagen nueva que se suba se optimiza automaticamente
- Se corrigio un error en los nombres de archivos WebP que impedia que se sirvieran

## Pagina de inventario (frontend)

- Los vehiculos ahora se ordenan alfabeticamente por marca (antes era aleatorio)
- Se agrego barra de herramientas moderna con opciones de ordenar: Marca A-Z, Marca Z-A, Precio menor a mayor, Precio mayor a menor, Año mas nuevo, Año mas antiguo
- Se agrego filtro por marca con dropdown
- Se agrego filtro por año con dropdown
- Se agrego buscador en tiempo real
- Se agrego boton "VER VENDIDOS" para alternar entre disponibles y vendidos
- Se agrego boton "Limpiar filtros" cuando hay filtros activos
- Se muestra contador de vehiculos visibles
- Todo funciona tanto en desktop como en celular

## Panel de administracion

- El estado "NO RESIDENTES" ahora se ve claramente en el listado de vehiculos sin necesidad de pasar el mouse por encima
- Los vehiculos NO RES tienen borde azul y fondo destacado para identificarlos rapido

## Resumen de mejoras

| Antes | Despues |
|-------|---------|
| Carga: 20.6s | Carga: 1.6s |
| Imagenes sin optimizar | 2,037 imagenes optimizadas + WebP |
| Sin cache | Cache de 1 anio para imagenes |
| Sin compresion | Gzip activo |
| NO RES dificil de ver | NO RES siempre visible |
| Inventario sin ordenar | Ordenado por marca + filtros + buscador |
